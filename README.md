# User-Directory-with-Live-Search
A simple JavaScript project that displays a list of users and allows real-time searching by name or email.

Link: https://ds1322.github.io/User-Directory-with-Live-Search/

🚀 Features
	•	Display dynamic user cards from a JavaScript array
	•	Live search functionality (updates as you type)
	•	Case-insensitive search
	•	DOM manipulation using JavaScript
	•	Array methods like filter() and map()
	•	Clean and responsive UI

🛠 Tech Stack
	•	HTML
	•	CSS
	•	JavaScript (Vanilla JS)
